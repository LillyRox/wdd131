FocusFlow: Site Plan

Lilly Armenta - WDD 131

Site Name

FocusFlow

This name reflects a smooth and uninterrupted workflow. It emphasizes helping users stay focused and maintain momentum while managing their daily responsibilities efficiently.

Site Purpose

FocusFlow is a lightweight task tracking application built as a Frontend MVP. Its main goal is to offer a minimalistic and responsive interface where users can quickly log, view, and manage their daily activities, with data stored locally in the browser.

Scenarios

  • "I just thought of something important I need to do later; how can I quickly save it so I do not forget even if I close my browser?"
  • "I have completed a task and want to keep my list clean; how can I easily remove it and track what is left?"

Color Schema

The palette is inspired by clarity and productivity:

Primary: #2C3E50 (Midnight Blue) - Used for headings and navigation.
Secondary: #ECF0F1 (Clouds) - Used for backgrounds and reading areas.
Accent: #27AE60 (Nephritis Green) - Used for action buttons and confirmations.

Typography

Montserrat: Used for headings to create a clean and modern visual hierarchy.

Open Sans: Used for body content to ensure readability across devices, especially on smaller screens.
