FocusFlow: Site Plan

Lilly Armenta - WDD 131

Site Name

FocusFlow

The name represents a smooth, uninterrupted workflow. It reflects the idea of staying focused while managing tasks efficiently in a clean and modern environment.

Site Purpose

FocusFlow is a lightweight task tracking application designed as a frontend MVP. It provides a modern, distraction-free interface where users can quickly add, manage, and remove tasks. All data is stored locally in the browser, ensuring fast performance and simplicity.

Scenarios

  • "I need a fast and simple way to save a task before I forget it."
  • "I want a clean interface where I can quickly remove completed tasks and stay organized."

Color Schema

The palette focuses on a modern, soft, and friendly interface:

Primary: #6C63FF (Modern Violet) – Used for branding, highlights, and navigation accents.
Secondary: #F5F7FF (Soft Background) – Used for the main background and layout areas.
Accent: #FF7A59 (Coral Orange) – Used for buttons and interactive elements.

Typography

Montserrat: Used for headings to create a strong and modern visual hierarchy.

Inter: Used for body text to improve readability and provide a clean, contemporary look.
